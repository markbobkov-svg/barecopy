What gets removed

Word, Excel, PowerPoint

Author, last modified by, company, manager, custom properties, template name, total editing time, print date and the embedded preview thumbnail — cleared from docProps inside the file.

PDF

Author, title, subject, keywords, creator and producer application, creation and modification dates — cleared from both the info dictionary and embedded XMP metadata.

Photos

All EXIF data: GPS coordinates, camera make and model, capture date, editing software. Metadata segments are cut out at byte level — pixels are copied untouched, so the clean copy is lossless. Works with full-size phone photos.

Honest limits

Tracked changes and comments in Word are content, not metadata — Barecopy detects them and warns you, but you should accept or reject them in Word before sending.

Verify it yourself

Reopen the clean copy and check File → Info, or right-click → Properties → Details. The identifying fields will be empty.

Zero servers

This page has no backend and sends no requests with your data. Open DevTools → Network while cleaning a file: nothing leaves.

Prove what you removed

Removing metadata is only half the job when a client, reviewer or court needs to know it was done. Barecopy Pro produces a verification report: a PDF listing every field removed from every file, the SHA-256 checksum of each file before and after cleaning, and the result of an automatic re-scan of the cleaned copy.

The before/after hashes are a chain-of-custody record — the recipient can confirm the exact file that was verified. No other metadata tool produces this.

Questions

How do I remove the author name from a Word document?

Drop the .docx above. Barecopy shows the author, "last modified by", company and every custom property, then produces a copy with all of them cleared — including fields Word's own Document Inspector misses on Mac. The file never leaves your device.

Does exporting Word to PDF remove metadata?

No. The author name, creator application and timestamps are carried over into the PDF's info dictionary and XMP block. Clean the PDF here after exporting, or clean the Word file before.

How is this different from uploading to an online metadata remover?

You are removing metadata precisely because the file is sensitive — sending it to someone's server defeats the point. Barecopy is a page of JavaScript that processes files inside your browser. Disconnect from the internet after loading the page: it still works.

Do cleaned photos lose quality?

No. Metadata segments (EXIF, GPS, IPTC, comments) are cut out at byte level and the image data is copied untouched — the clean copy is pixel-identical, with the color profile preserved.

What does Barecopy not remove?

Tracked changes and comments in Word are document content, not metadata. Barecopy detects them and warns you; accept or reject them in Word before sending. Password-protected PDFs are refused rather than risk corrupting them.

What exactly does the Pro verification report contain?

A plain-text record generated locally: for each file, every metadata field that was found and removed, any warnings, and the SHA-256 checksum of the clean copy — useful when you need to show a client or a court exactly what was scrubbed and prove which file version you sent.

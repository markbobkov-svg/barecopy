# Barecopy — ранбук деплоя (~40 минут твоего времени)

Пакет: `index.html` (приложение + лендинг + Pro), `privacy.html`, `terms.html`,
`robots.txt`, `sitemap.xml`, `favicon.svg`, `og.png`, `vercel.json`.
Тесты: 86/86 (логика 50 + целостность файлов 27 + Pro-слой 9).

⚠️ Никогда и никому (включая меня) не пересылай ключи, токены и пароли.
Всё секретное вводится только тобой в дашбордах сервисов.

---

## Шаг 1 — Домен (5 мин, €9.65/год)
1. Купи `barecopy.com` (Cloudflare Registrar / Porkbun / Namecheap — где смотрел цену).
2. Только купить. DNS настроим на шаге 3. Домен уже вшит в файлы пакета.

## Шаг 2 — GitHub (5 мин)
1. github.com/new → имя `barecopy`, Public или Private — неважно → Create.
2. «uploading an existing file» → перетащи ВСЁ содержимое zip (не сам zip) → Commit.

## Шаг 3 — Vercel (5 мин)
1. vercel.com → Sign up with GitHub.
2. Add New → Project → Import `barecopy` → Deploy (настройки не трогать —
   это статический сайт).
3. Получишь адрес вида `barecopy.vercel.app` — уже можно открыть и потыкать.
4. Project → Settings → Domains → добавь купленный домен → Vercel покажет
   1–2 DNS-записи → внеси их у регистратора. Подхватится за минуты, максимум пару часов.

## Шаг 4 — Polar (15 мин) — деньги
1. polar.sh → Sign in with GitHub → Create organization (имя: Barecopy).
2. Products → New Product:
   - Name: `Barecopy Pro`
   - Pricing: Subscription → Monthly, **€4/мес**
   - Benefits → добавь **License Keys** (activations: unlimited) → Save.
     Ключ привязан к подписке: при отмене Polar отзывает его в конце
     оплаченного периода — сайт это увидит при суточной проверке и выключит Pro.
3. Checkout Link не появляется сам — создай его: на странице продукта раздел
   **Checkout Links** → **Create Checkout Link** → выбери `Barecopy Pro` →
   Create → скопируй сгенерированную ссылку (вида `buy.polar.sh/...`).
4. Скопируй **Organization ID** (Settings организации).
5. GitHub → открой **config.js** (единственный файл, который ты правишь руками)
   → карандаш (Edit) → впиши между кавычками:
   - `window.BARECOPY_ORG_ID`   = твой Organization ID
   - `window.BARECOPY_CHECKOUT` = твоя buy.polar.sh/... ссылка
   Commit → Vercel передеплоит сам (~1 мин).
   ⚠ Это делается ОДИН раз. Дальше при любом обновлении кода config.js
   остаётся нетронутым — просто качаешь новый билд и коммитишь, правок нет.
6. Для выплат Polar попросит онбординг Stripe: паспорт + IBAN.
   Финляндия поддерживается. Цепочка денег: покупатель → Polar (merchant of
   record, EU VAT — их проблема) → выплата на твой счёт.

## Шаг 5 — Смоук на проде (5 мин)
1. Оформи подписку сам (потом отмени + refund из дашборда Polar).
2. Ключ из письма → вставь на сайте → Activate.
3. Накидай 4+ файлов → ZIP + report → открой `barecopy-report.txt`.
4. Проверь отмену: после cancel ключ должен перестать проходить проверку
   (максимум через сутки после конца периода).
5. Если активация выдаст ошибку — пришли мне ТЕКСТ ошибки (не ключ):
   форма Polar API могла измениться, правится одной функцией `validateLicense`.

Известные ограничения v1.1 (осознанные, не баги):
- Tracked changes / комментарии Word — детектим и предупреждаем, не удаляем.
- Запароленные PDF — отклоняем с инструкцией (иначе файл ломается).
- HEIC пока не поддержан (конвертится айфоном при выборе через браузер).
- Лицензия хранится в localStorage браузера — смена браузера = повторная активация.
- Подписка перепроверяется раз в сутки; офлайн-grace 7 дней, потом Pro ставится на паузу до первой проверки.
- Смена цены: Polar (сам продукт) + index.html — карточка Pro (2 места) и JSON-LD (1 место).
